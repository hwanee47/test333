APP 이용안내
CJ대한통운 화물정보망의 첨단 물류 IT 시스템으로 차별화된 물류 서비스를 경험하실 수있습니다.
차주용 이용안내 화주용 이용안내 파트너사 이용안내

주요기능 요약 안내

  • - 화물조회, 운행보고, 길안내 서비스 등 차주용 모바일 업무에 적합한 서비스를 제공합니다.
  • - 간단한 절차로 서비스 이용이 가능합니다.
  • - 모바일 앱을 통해 알림내역 환경설정 등 다양한 편의서비스를 제공합니다.
  • 01. 앱 설치 후 최초 1회 모바일 앱 접근 관련안내 동의
  • 02. 정(준)회원일 경우 핸드폰번호, 차량번호를 입력 후 로그인을 할 경우 메인화면으로 이동
  • 03. 각 영역 선택 시 "내화물 조회", "공개화물 조회", "추천화물 조회" 이용
  • 04. 화물 조회 해당 물량 선택 시 "운행보고" 이용
  • 05. 운행보고 상세(단계별 보고)
  • 06. 해당 영역 선택 시 "공차관리" 이용(현위치, 지역선택, 예약기능)
  • 07. 공차 관리 버튼이 OFF인 상태에서 선택
  • 08. 완료 시에 공차 버튼 ON으로 변경
  • 09. 환경설정 버튼을 선택 시 환경설정 화면으로 이동
  • 10. 음성안내, 네비게이션, 자동실행, 알림설정, 자동업데이트, 설정 가능
  • 11. 설치정보(QR 코드 혹은 URL로 접속하여 설치가능)

주요기능 요약 안내

  • - 간편인증, 화물등록, 운송현황 조회 등 화주용 모바일 업무에 적합한 서비스를 제공합니다.
  • - 간단한 절차로 서비스 이용이 가능합니다.
  • - 모바일 앱을 통해 실시간 운송현황 정보를 제공합니다.
  • 01. 앱 설치 후 최초 1회 모바일 앱 접근 관련안내 동의
  • 02. 정(준)회원일 경우 아이디, 비밀번호를 입력후 로그인을 할 경우 메인화면으로 이동
  • 03. 환경설정에서 간편인증을 설정한 경우 지문인증을 통한 로그인 처리 후 메인화면으로 이동
  • 04. 화물등록시 등록상세 페이지로 이동(계약화물, 계약외 화물 간편등록 기능제공)
  • 05. 해당 영역 선택 시 "화물조회" 이용
  • 06. 각 운송단계별 현황 제공
  • 07. 구글지도를 통한 정확한 운송위치 정보 제공
  • 08. 환경설정 버튼을 선택 시 환경설정 화면으로 이동
  • 09. 음성안내, 네비게이션, 자동실행, 알림설정, 자동업데이트, 설정 가능
  • 10. 설치정보(QR 코드 혹은 URL로 접속하여 설치가능)

주요기능 요약 안내

  • - 간편인증, 배정물량 배차, 소속차량 관리 등 파트너사용 모바일 업무에 적합한 서비스를 제공합니다.
  • - 간단한 절차로 서비스 이용이 가능합니다.
  • - 모바일 앱을 통해 쉽고 간편한 배차확정(취소) 서비스를 제공합니다.
  • 01. 앱 설치 후 최초 1회 모바일 앱 접근 관련안내 동의
  • 02. 파트너 회원일 경우 아이디, 비밀번호를 입력 후 로그인을 할 경우 메인화면으로 이동
  • 03. 환경설정에서 간편인증을 설정한 경우 지문인증을 통한 로그인 처리 후 메인화면으로 이동
  • 04. 메인화면에서 배정물량에 대한 배차확정 처리
  • 05. 차량 검색 시 소속차량 관리 및 등록기능 제공
  • 06. 운송현황 버튼을 선택 시 단계별 운송현황 조회화면으로 이동
  • 07. 팝업 메뉴 선택 시 팝업메뉴 화면으로 이동
  • 08. 업무별 메뉴로 이동이 가능하며 그 외 환경설정 및 이용안내화면 선택 시 해당 조회 화면으로 이동
  • 09. 나의 알림 선택 시 알림 화면으로 이동
  • 10. 주요 알림 내역을 확인하실수 있으며 보관주기는 일주일입니다.
  • 11. 설치정보(QR 코드 혹은 URL로 접속하여 설치가능)
