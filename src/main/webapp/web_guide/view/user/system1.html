권한별 메뉴관리

메뉴별 권한을 설정하는 화면입니다.
관리자가 직접 사용자마다 메뉴를 다르게 나타나도록 설정할 수 있습니다.

1. 신규 권한 생성하기
> STEP-1 새로운 권한메뉴추가를 원하실경우 상단의 신규(F3)버튼을 클릭하여 행을 추가합니다.
> STEP-2 행이 추가되면 그리드의 노란색필드(필수)에 값을 입력합니다.
> STEP-3 입력을 완료하셨으면 상단의 저장(F4)버튼을 클릭하여 저장합니다.
2. 생성된 권한에 메뉴 부여하기
> STEP-1 생성한 권한 목록중 그리드에서 마우스로 선택합니다.
> STEP-2 권한을 선택하면 하단의 메뉴리스트에 부여가능한 메뉴가 나타나며 부여할 메뉴를 체크합니다.
> STEP-3 오른쪽(>) 화살표 버튼을 클릭하면 체크된 메뉴가 부여됩니다.
3. 부여된 메뉴 제거하기
> STEP-1 생성한 권한 목록중 그리드에서 마우스로 선택합니다.
> STEP-2 권한을 선택하면 하단의 권한 메뉴리스트에 부여된 메뉴목록이 나타나며 권한에서 제거할 메뉴를 체크합니다.
> STEP-3 왼쪽(<) 화살표 버튼을 클릭하면 체크된 메뉴가 권한에서 제거됩니다.
4. 조회하기
> STEP-1 조회목록의 필터링을 원하실땐 조회조건을 입력합니다.
> STEP-2 상단의 조회(F2)버튼을 클릭하여 조회합니다.
5. 사용안함 처리하기
일시적인 사용을 막으려면 사용안함으로 처리합니다.
> STEP-1 상단의 조회(F2)버튼을 클릭하여 대상을 조회합니다.
> STEP-2 대상을 그리드에서 체크한뒤 사용여부필드를 "사용안함"으로 변경합니다.
> STEP-3 상단의 저장(F4)버튼을 클릭하여 저장합니다.
6. 삭제하기
사용을 막고 목록에서 제거하려면 삭제 처리합니다.
> STEP-1 상단의 조회(F2)버튼을 클릭하여 대상을 조회합니다.
> STEP-2 대상을 그리드에서 체크한뒤 상단의 삭제(F5)버튼을 클릭하여 대상을 삭제합니다.
