2. 사용자 유형

GWDS는 사용자 유형(SM998)을 다음과 같이 정의하고 있습니다.

코드 설명
SUPER 시스템관리자용
ADMIN 업체직원용(시스템관리자)으로 시스템계약시 최초 계정1개 부여
M01 업체직원용(창고&배송담당자)으로 시스템관리자(ADMIN)가 관리
M02 업체직원용(창고담당자)으로 시스템관리자(ADMIN)가 관리
M03 업체직원용(배송담당자)으로 시스템관리자(ADMIN)가 관리
U01 업체직원용(외부사용자)으로 시스템관리자(ADMIN)가 관리 - 화주(물건의 주인) 또는 발송고객이 이에 해당.
3. 비밀번호

관리자가 계정등록시 임시비밀번호는 기초코드(SM1001)의 코드명에 정의되어 있으며 해당 계정 사용자가 접속시 비밀번호를 변경합니다.
최초 1번 변경가능하며 이후 변경을 원할경우 [사용자 관리]화면에서 초기화기능으로 변경가능합니다.
변경시 비밀번호의 유효성 체크사항은 다음과 같습니다.

  • 비밀번호와 비밀번호 재확인 비교
  • 공백 문자 사용 금지
  • 비밀번호는 8자 이상
  • 숫자 +영문+특수문자의 조합
  • 아이디와 4자리 이상 동일한 비밀번호 사용 금지
  • 비밀번호에 4자리 이상 반복되는 문자나 숫자 사용 금지
