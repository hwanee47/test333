고객사 상세관리

기등록된 배송 고객사의 발송고객 & 계약을 관리하는 화면입니다.

1. 조회하기
> STEP-1 조회목록의 필터링을 원하실땐 조회조건을 입력합니다.
> STEP-2 상단의 조회(F2)버튼을 클릭하여 조회합니다.
2. 거래구분 저장하기
> STEP-1 고객사 목록에서 대상을 선택한 후 기본정보의 [거래처구분]필드에 값을 선택합니다.
> STEP-2 상단의 저장(F4)버튼을 클릭하여 저장합니다.
3. 발송고객 등록하기
> STEP-1 고객사 목록에서 대상을 선택한 후 "발송고객등록" 버튼을 클릭합니다.
> STEP-2 상단의 신규버튼을 클릭하여 행을 추가하고 발송고객 정보를 입력합니다.
> STEP-3 상단의 저장버튼을 클릭하여 저장합니다.
4. 계약 등록하기
> STEP-1 [고객사 상세관리]화면 또는 [발송고객등록]화면에서 대상을 선택한 후 "계약등록" 버튼을 클릭합니다.
> STEP-2 상단의 신규버튼을 클릭하여 행을 추가하고 계약 정보를 입력합니다.
> STEP-3 상단의 저장버튼을 클릭하여 저장합니다.
5. 계약품목 등록하기
> STEP-1 [계약등록]화면에서 "계약품목" 버튼을 클릭합니다.
> STEP-2 상단의 신규버튼을 클릭하여 행을 추가하고 품목 정보를 입력합니다.
> STEP-3 상단의 저장버튼을 클릭하여 저장합니다.
6. 계약단가 등록하기
> STEP-1 [계약등록]화면에서 "계약단가" 버튼을 클릭합니다.
> STEP-2 상단의 신규버튼을 클릭하여 행을 추가하고 계약단가 정보를 입력합니다.
> STEP-3 상단의 저장버튼을 클릭하여 저장합니다.
7. 계약 갱신하기
> STEP-1 [계약등록]화면에서 "계약갱신" 버튼을 클릭합니다.
> STEP-2 나타나는 팝업창에서 확인버튼을 클릭하면 현재 계약 종료일을 기준으로 계약기간이 1년 연장됩니다.
